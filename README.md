# covid-19-india-data-analysis-dashboard
COVID-19 India data analytics project using SQL ETL and Excel dashboarding. Analyzed cases, testing, vaccination trends, and risk levels through data cleaning, transformation, KPI analysis, and interactive visualizations to generate actionable public health insights.

🦠 COVID-19 India Data Analysis Dashboard
📌 Project Overview

The COVID-19 pandemic generated massive amounts of healthcare data related to infections, testing, recoveries, fatalities, and vaccination progress.

This project analyzes COVID-19 trends across India using a complete data analytics workflow involving SQL ETL, Excel data analysis, and interactive dashboard visualization.

The objective was to transform raw pandemic data into meaningful insights that can support public health monitoring and decision-making.

🔍 Major Insights
1. COVID Cases Were Highly Concentrated in Specific States

Maharashtra, Karnataka, and Tamil Nadu accounted for a significant share of total reported cases.

2. Multiple COVID Waves Were Identified

Monthly trend analysis showed clear surges and declines corresponding to different pandemic waves.

3. Testing Capacity Influenced Detection Rates

Regions with lower testing coverage often recorded higher positivity rates, suggesting possible underreporting.

4. Vaccination Supported Better Recovery Trends

Higher vaccination coverage correlated with improved recovery outcomes and reduced severity.

💼 Business Recommendations
Increase testing in regions with high positivity rates.
Prioritize healthcare resources in high-risk areas.
Improve vaccination outreach in low-coverage regions.
Strengthen data collection pipelines for future health crises.
